Login Instructions:
Enter your registered phone number and password.
Ensure your phone number and password match the credentials used in the previous AIMI version. There is no need to add a new phone number.
After logging in, submit your version code to activate your account.

Submit your security fee to generate your version code.
Once submitted, the system will generate a verification code automatically. Paste this code to activate your account.
